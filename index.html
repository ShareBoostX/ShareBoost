Privacy Policy

ShareBoost Discord Bot

Effective Date: 25/10/2025

1. Information We Collect

  • ShareBoost only logs when someone joins or leaves a server, without storing their username or ID.
  • It tracks only the number of messages sent for statistics, without recording message content or authors.
  • No other personal information, such as emails or profile pictures, is collected.

2. How We Use Information

  • The collected data is used solely for server activity statistics and to run the automatic server promotion every 4 hours.
  • Data is not used for marketing or shared with any third parties.

3. Data Storage and Security

  • All stored data is secure and does not include personal information.
  • Data is never shared externally and is only used for the bot's intended functions.

4. Third-Party Services

  • ShareBoost does not send or share data with any external services.

5. User Rights

  • Users do not need to provide any information to the bot.
  • Removing the bot from a server immediately stops any limited data collection.

6. Changes to This Policy

  • Any updates to this privacy policy will be announced in the same place and take effect upon posting.
